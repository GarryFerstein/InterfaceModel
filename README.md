

1. **Описание проекта**:
   -  Интерактивное веб-приложение на Streamlit, позволяющее: 
- обучать модель машинного обучения на данных о ноутбуках,
- предсказывать цену как вручную, так и по загружаемому CSV-файлу.

-  Модель основана на CatBoostRegressor и учитывает как числовые, так и категориальные признаки (бренд, процессор, ОЗУ, накопитель, дисплей и др.).


2. **Функциональность**:
   
   - Обучение модели: загрузка обучающего датасета, настройка гиперпараметров CatBoost, оценка качества (R², MAE).

   - Инференс вручную: ввод характеристик ноутбука через форму → мгновенное предсказание цены.

   - Массовый инференс: загрузка CSV-файла с несколькими записями → получение предсказаний и возможность скачать результат.


3. **Гиперпараметры модели**:

   - Количество итераций (iterations)

   - Скорость обучения (learning_rate)

   - Глубину деревьев (depth)

   - Случайное зерно (random_seed)


4. **Установка**:
   - 1. Клонируйте репозиторий:
   https://github.com/GarryFerstein/InterfaceModel
   - 2. Настройка виртуального окружения:
   python -m venv venv
   source venv/bin/activate  # На Windows: venv\Scripts\activate
   - 3. Установка зависимостей:
   pip install -r requirements.txt
